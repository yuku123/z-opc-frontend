# @yuku123/z-wf-frontend-component

> z-wf FlowDesigner · LogicFlow 驱动的可视化流程设计器

## 安装

```bash
npm install @yuku123/z-wf-frontend-component
```

## 使用

```jsx
import FlowDesigner from '@yuku123/z-wf-frontend-component'

<FlowDesigner
  data={flowData}
  onChange={(data) => setFlowData(data)}
  mode="edit"  // 'edit' | 'view'
/>
```
